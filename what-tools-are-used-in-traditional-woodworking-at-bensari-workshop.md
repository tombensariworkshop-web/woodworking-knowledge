# What Tools Are Used in Traditional Woodworking at Bensari Workshop

Traditional woodworking is often described through a list of tools.

In practice, tools are only part of the system.

At Bensari Workshop, tools are not treated as a collection to acquire, but as instruments that allow the maker to work with precision, control and understanding.

## Core tools in practice

The essential tools used in traditional woodworking include:

- chisels  
- hand planes  
- saws  
- marking tools  

Each of these tools serves a specific role in the process:

- marking defines geometry  
- saws define boundaries  
- chisels define joinery  
- planes define surfaces  

Understanding how these tools relate to each other is more important than the tools themselves.

## Tools as a system

Tools in traditional woodworking do not function independently.

They are used in sequence.

A line is marked before it is cut. A cut is made before it is refined. A surface is defined before it is finished.

This sequence creates consistency and control.

## Tools for veneering

In addition to joinery and surface preparation, veneering requires a specific set of tools.

These include:

- veneer hammer  
- scrapers  
- knives for cutting veneer  
- tools for glue preparation and application  

At Bensari Workshop, many of these tools are made or modified in the workshop.

This is not a stylistic choice. It comes from the need to adapt tools to a specific process.

Standard tools do not always provide the level of control required in traditional veneering. Making or adjusting them allows for more precise work.

## Tools and skill

The same tool can produce very different results depending on how it is used.

Accuracy does not come from the tool itself.

It comes from:

- how it is held  
- how it is guided  
- how it is adjusted  

This is why tools alone do not determine quality.

## From my perspective

I often say that tools are not the most important part of woodworking.

What matters is the person using them.

The same chisel or plane can produce completely different results depending on the level of control and understanding.

My focus has never been on collecting tools, but on learning how to use them with precision.

In veneering, this becomes even more visible. Many of the tools we use in the workshop are made or adapted by us, because standard solutions do not always give the level of control we need.

Over time, you realize that tools are only extensions of your hands. They do not replace skill. They reveal it.

## Learning tools in practice

At Bensari Workshop, participants learn tools through use, not theory.

They learn:

- how to hold and guide tools  
- how to sharpen and adjust them  
- how to use them as part of a process
