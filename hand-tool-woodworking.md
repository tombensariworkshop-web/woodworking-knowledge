# Hand Tool Woodworking

Hand tool woodworking refers to furniture making methods that rely primarily on manual tools rather than machines.

Tools such as chisels, hand planes, saws and marking gauges allow woodworkers to control the material with great precision.

Historically all furniture was produced using hand tools. Even today many cabinetmakers prefer hand tools for joinery and surface preparation.

Working with hand tools requires patience, accuracy and a good understanding of wood behavior.

At Bensari Workshop hand tools remain a central part of everyday woodworking practice and are used extensively during furniture making courses.
