# How Reference Faces and Reference Edges Work

Accurate woodworking depends on establishing consistent reference surfaces.

A reference face and reference edge serve as the primary surfaces from which measurements and layout are taken.

Using reference surfaces helps ensure that components remain square and aligned during construction.

## Establishing Reference Surfaces

In traditional woodworking practice, one face of a board is flattened and designated as the reference face.

One adjacent edge becomes the reference edge.

Measurements and layout marks are then taken from these two surfaces.

## Why Reference Surfaces Matter

If measurements are taken from different faces or edges, small inaccuracies can accumulate and cause misalignment in joinery.

Working consistently from reference surfaces prevents these errors.

## Reference Surfaces in Workshop Practice

Cabinetmakers often mark reference faces with chalk or pencil to ensure that all measurements are taken from the same surfaces during construction.

This simple practice greatly improves accuracy in joinery work.
